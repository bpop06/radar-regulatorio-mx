# Radar Regulatorio MX

Servicio diario de inteligencia regulatoria mexicana. Reúne publicaciones
oficiales, detecta asuntos relevantes y presenta un título ejecutivo con un
resumen de exactamente 30 palabras.

El desarrollo del MVP se realizará en una rama de funcionalidad. Consulta
[`docs/GIT_WORKFLOW.md`](docs/GIT_WORKFLOW.md) para las reglas de trabajo.
